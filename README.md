# PROVA ESCOLA DE TI

## Back End

Utilizado o Nest.JS e MongoDB

1. Pasta Back End

   ```bash
   cd .\back-end\
   ```

2. Instalando Dependencias

   ```bash
    npm install
   ```
2. Compilando e Executando o Projeto

   ```bash
    npm run start
   ```

## Front End

Utilizado o React Native com Expo

1. Pasta Front End

   ```bash
   cd .\front-end\
   ```

2. Instalando Dependencias

   ```bash
    npm install
   ```

2. Executando o Projeto

   ```bash
    npx expo start --web
   ```
